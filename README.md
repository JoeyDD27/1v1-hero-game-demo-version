# 1v1 Hero Support Spell Game - Demo

A 2D top-down local multiplayer battle game built with Godot 4.x.

## Setup

1. **Install Godot 4.x**: Download from https://godotengine.org/download
2. **Open Project**: Open this folder in Godot
3. **Run**: Press F5 or click the Play button

## How to Play

### Single Player Test (Quick Start)
- Click "Quick Start" button on main menu
- Test mechanics without networking

### Multiplayer (2 Computers on Same WiFi)

**Player 1 (Host):**
1. Click "Host Game"
2. Share your IP address (shown on screen)
3. Wait for Player 2 to connect

**Player 2 (Client):**
1. Click "Join Game"
2. Enter Player 1's IP address
3. Click "Connect"

## Controls

- **WASD**: Move hero
- **Mouse**: Aim/Attack (auto-attacks nearest enemy)
- **Q**: Ability 1
- **E**: Ability 2
- **1/2/3**: Summon supports
- **R**: Cast big spell
